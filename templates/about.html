About Movie Recommender

A semantic content-based movie recommendation system built using Machine Learning, Natural Language Processing, Flask and Python.

What is this?

Movie Recommender is a semantic content-based recommendation system that helps users discover movies similar to the ones they already enjoy. Instead of relying on popularity or user ratings, the recommendation engine analyzes movie metadata such as genres, keywords, cast, director and plot descriptions to understand the overall context of a movie and recommend titles with similar themes and content.

How it Works

  • Search for any movie available in the database.
  • Movie metadata is transformed into semantic embeddings using Sentence-BERT (SBERT).
  • Cosine Similarity compares the semantic representations of every movie.
  • The five most relevant movies are recommended.
  • Detailed movie information is retrieved through the TMDB API.

Why Sentence-BERT?

Traditional text vectorization techniques such as Bag of Words and TF-IDF rely primarily on word occurrence and frequency. While they provide a strong baseline, they cannot understand the contextual meaning of text. As a result, two movies describing similar ideas using different words may appear unrelated.

Sentence-BERT (SBERT) generates dense semantic embeddings that capture the meaning of an entire movie description rather than simply matching common words. This enables the recommendation engine to identify movies with similar themes and context, resulting in more meaningful recommendations.

Features

  • Semantic movie recommendations using Sentence-BERT embeddings.
  • Detailed movie information including poster, genres, runtime, director, cast and overview.
  • Popular Movies section on the home page.
  • Search suggestions for incorrect or incomplete movie titles.
  • Live movie metadata retrieved using the TMDB API.
